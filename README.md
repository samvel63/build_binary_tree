# Лабораторная работа №23. Построение бинарного дерева. #
## Вариант 5. Определить значение нетерминальной вершины дерева с максимальной глубиной. ##

***

### Основные понятия: ###
<li>**Глубиной вершины** дерева называется длина пути в эту вершину из корня.
<li>**Листом** или **терминальной вершиной** дерева называется вершина, не имеющая поддеревьев.
<li>**Нетерминальной вершиной** называется родитель **терминальной вершины**.
***

### Алгоритм решения задачи: ###
1. Считываем последовательность чисел, строя из них упорядаченное бинарное дерево.
2. Ищем терминальные вершины.
3. Найдя терминальную вершину, ищем её родителя(нетерминальную вершину) и добавляем в массив нетерминальных вершин.
4. Находим глубину каждой нетерминальной вершины и добавляем в массив глубин нетерминальных вершин.
5. Находим максимальную глубину и выводим значение вершин с найденной глубиной.
